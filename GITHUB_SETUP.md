
### 3. Connect to GitHub
```bash
git remote add origin https://github.com/YOUR_USERNAME/flightguard-dapp.git
git branch -M main
git push -u origin main
```

Replace `YOUR_USERNAME` with your GitHub username.

### 4. Verify
- Go to your GitHub repository
- Check that files are uploaded
- Verify `.gitignore` is working (no node_modules, venv visible)

### Files EXCLUDED from Git (in .gitignore):
- `node_modules/` - Will be installed by setup.bat
- `venv/` - Will be created by setup.bat
- `*.log` files - Generated at runtime
- `.env` - Contains sensitive data
- `db.sqlite3` - Will be created fresh
- `cache/`, `artifacts/` - Generated by Hardhat

### Files INCLUDED in Git:
- All source code (.js, .jsx, .py, .sol)
- Configuration files (package.json, requirements.txt)
- Documentation (.md files)
- Scripts (.bat files)
- `deployed_addresses.json` - Blockchain config
- `.gitignore`, `.gitkeep` files
